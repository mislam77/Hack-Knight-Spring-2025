## Getting Started
### Client
```
cd client
npm install --legacy-peer-deps
npm run dev
```
### Server (for Windows)
```
cd server
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
flask --app server --debug run
```
